# Validação de Login

## Tecnologias:

*Node.js</br>
*express</br>
*uuid</br>
*cors</br>

## Sobre:

Api de login de usuario, onde valida os dados dos usuarios não aceita login duplicado, colocamos a porta da pagina Front-end, e mandamos os dados para 
o banco de dados e ainda tem a opção de delete
